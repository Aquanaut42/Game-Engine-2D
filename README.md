# Look-Around-World

This is a test, to build a basic graphics for a world, allows the user to look around the world

(Based on the Forest-Fire-Simulation code, with a lot of modification)
(You can run it by running the engine file)
(to rebuild it run this command in the terminal in the folder
g++ main.cpp logic.cpp window.cpp -lSDL2 -lSDL2_image -lSDL2_ttf -o engine )
